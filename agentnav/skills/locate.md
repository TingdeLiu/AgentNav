# locate skill

当需要导航到某个目标时，使用以下步骤将目标定位为可导航位姿：

## 步骤

1. **robot_capture()** — 获取当前相机帧，直接观察场景

2. **估算目标像素坐标** — 在图中找到目标中心，估算其像素坐标 (u, v)：
   - u：从图像左边缘起的水平像素数
   - v：从图像顶边缘起的垂直像素数
   - 典型图像分辨率为 640×480，画面中心约为 (320, 240)
   - 如果目标在画面右侧偏下，u > 320, v > 240

3. **pixel_to_pose(u, v)** — 将像素坐标转换为机器人坐标系位姿
   - 返回 `{x, y, theta}`，单位：米 / 弧度
   - 需要深度相机在 /camera/depth 上发布数据

4. **s1_move(pose)** — 将位姿交给 S1 执行导航，返回 task_id

## 注意事项

- 若 pixel_to_pose 返回 `{"error": "No depth frame available"}`，说明深度相机未就绪，检查 /camera/depth 话题
- 若目标估算偏差导致 s1_move 到错误位置，重新 robot_capture → 重新估算 → 重试
- 目标距离越近，深度精度越高；建议在 0.5m–5m 范围内使用

## 示例

```
robot_capture()
→ [看到黑色椅子，位于画面中右偏下约 60% 宽、45% 高处]
→ u ≈ 384, v ≈ 216

pixel_to_pose(384, 216)
→ {"x": 1.8, "y": -0.3, "theta": 0.0}

s1_move({"x": 1.8, "y": -0.3, "theta": 0.0})
→ task_id=a1b2
```
